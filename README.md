# Introduction
This repository is a seed to start an Angular2 meteor project with authentication, pagination and Angular2 Material.

Includes :
* Meteor 1.4.2
* Angular 2
* Typescript configuration
* Angular2 Material
* Simple Todo application
* Authentication + User Sign up
* Pagination
* Searching by name
